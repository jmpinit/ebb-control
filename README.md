# ebb-control

![Test status](https://github.com/jmpinit/ebb-control/actions/workflows/test.yml/badge.svg)

WebSerial interface library for the EiBotBoard - the controller in the AxiDraw,
Egg-Bot, and WaterColorBot.

Used by the [p5.axidraw](https://github.com/jmpinit/p5.axidraw) library, which allows an
[AxiDraw](https://axidraw.com/) to be controlled from [p5.js](https://p5js.org/).

## Documentation

See [jmpinit.github.io/ebb-control](https://jmpinit.github.io/ebb-control) for the API docs.
